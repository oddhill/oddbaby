# Odd baby

A starter theme for Drupal theming.

### Dependencies
* Grunt (included)
* Sass 3.4
* Scss lint

### Includes
* Bourbon
* Neat
* Modernizr - Support for csstransform 2D & 3D, svg touch & includes SHIV.
* Selectivizr
