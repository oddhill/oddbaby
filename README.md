# Odd baby
> A starter theme for Drupal theming.

## Usage
```
npm install
```

Gulp tasks:
```
// Watch .scss (scss-lint, sass) and .js (jshint)
$ gulp watch

// Svg2png
$ gulp svg2png
```

## Dependencies
* Gulp
* Sass 3.4
* Scss lint

## Includes
* Bourbon
* Neat
* Modernizr - Support for csstransform 2D & 3D, svg touch & includes SHIV.
* Selectivizr
